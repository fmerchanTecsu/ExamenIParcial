Examen Auditoria de Software. Instituto Tecnológico Sudamericano Git y Github

Alumn@: JESUS JEANCARLOS ZAMBRANO MARTINEZ

  1. Qué es un sistema de control de versiones?

    Se llama control de versiones a la gestión de los diversos cambios que se realizan sobre los elementos de algún producto o una configuración del mismo.

  2. Cuál es la diferencia entre git y github?

    La diferencia es que git es una herramienta de codigo abierto para los desarrolladores para gestionar su codigo fuente y git hub es un servicio en linea al que los desarrolladores que utilizan Git pueden conectarse y cargar o descargar recursos.

  3. Cómo creo una rama en git?

    Para crear una rama en git se utiliza el comando git branch.

  4. Cómo clono un repositorio?

    Se lo clona con el comando git clone y el link del repositorio.

  5. Qué es la auditoría de software?

    La auditoria de software consiste en identificar los programas informáticos que utiliza una organización. La recopilación de información del software proporciona un historial de cambios. Es una medida preventiva para las empresas para evitar el uso fraudulento de software.

  6. Qué características considera importante para evaluar la facilidad de instalación de un software

    Que cumpla con su objetivo, que sea facil de usar, que tenga una buena seguridad, que se pueda modificar por otros desarrolladores y que se pueda utilizar en diversos equipos.

  7. Qué es un repositorio?

    Es un almacenamiento virtual de tu proyecto. Te permite guardar versiones del código a las que puedes acceder cuando lo necesites.

  8. Cuál es la finalidad de la norma ISO 27000

    Es un conjunto de estándares internacionales sobre la Seguridad de la Información. La familia ISO 27000 contiene un conjunto de buenas prácticas para el establecimiento, implementación, mantenimiento y mejora de Sistemas de Gestión de la Seguridad de la Información.

  9. Cómo le digo a git cual es mi email y correo?

    Con el comando git config --list

  10. Cuales son los pasos para conectar git con github?

    Descarga e instala la última versión de Git. Configura tu nombre de usuario en Git. Configura tu dirección de correo electrónico de confirmación en Git.
