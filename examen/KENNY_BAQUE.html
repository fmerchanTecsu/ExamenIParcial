  1. Qué es un sistema de control de versiones?

    Es donde se guarda y se puede editar proyectos de forma remota en el desarrollo de proyectos

  2. Cuál es la diferencia entre git y github?

    git nos permite el cambio de forma local mientras que github nos permite subir el proyecto y poder trabajarlo de forma colaborativa

  3. Cómo creo una rama en git?

    para crear una rama solo se utiliza git branch NOMBRE_DE_RAMA o tambien se puede usar git checkout -b NOMBRE_DE_LA_RAMA para que se abra directamente la rama creada

  4. Cómo clono un repositorio?

    se utiliza git clone ENLACE_DE_REPOSITORIO previamente abierto el gitbash en la carpeta donde se desea clonar

  5. Qué es la auditoría de software?

    es la practica que se utiliza para medir y actualizar los sistemas de empresas

  6. Qué características considera importante para evaluar la facilidad de instalación de un software

    aspectos importantes en la isntalacion seria que tenga un README.txt de la forma de instalacion ya sea por defecto o con opciones avanzadas

  7. Qué es un repositorio?

    un repositorio es donde se guarda o almacena un proyecto enviado desde el git

  8. Cuál es la finalidad de la norma ISO 27000

    asegurar la confidencialidad e integridad de los datos y de la informacion
